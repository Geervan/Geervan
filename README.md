# 👾 who am I

I'm **Geervan**, a computer science student who enjoys building software experiments.

I’m less interested in **cloning the same SaaS dashboard 100 times**, and more interested in exploring strange or creative ideas in software:

* chaotic cryptography simulations
* automation bots
* AI-assisted developer tooling
* interactive 3D environments
* small tools that solve oddly specific real-world problems

Most of my projects start as **“what happens if I try this?”** experiments.

---

# 🧪 projects / experiments

### 🧊 Rubik Cube Cryptography Simulator

A cryptographic experiment where a **3D Rubik's Cube acts as a dynamic symmetric encryption key**.

Instead of traditional key schedules, the cube's chaotic transformations drive encryption states.
Built using **Three.js + WebRTC for real-time encrypted messaging experiments.**

---

### 🤖 GitHub PR Review Bot

A developer workflow bot that analyzes pull requests and generates **structured review feedback automatically**.

Idea: reduce the friction of manual code review using AI-assisted analysis.

---

### 🌌 Cosmos Classroom

An experiment in **3D web learning environments**.

A small virtual campus built with **React + Three.js** exploring how spatial environments could make studying more immersive.

---

### 🛰 SpaceBasic Bot

Automation bot designed to streamline **hostel housekeeping requests and attendance tracking** by interacting with the student management system.

Built mainly with **Python automation + Playwright**.

---

### 📱 Class Item Reminder

A **Flutter mobile tool** built to solve a very specific real-world annoyance:

students forgetting items after class.

Uses **local notifications + checklist workflows** instead of traditional alarms.
